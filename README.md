# Hướng dẫn sử dụng

```
> git clone https://github.com/ttkienn/spam_cmt_Facebook
> cd spam_cmt_Facebook
> Sửa nội dung trong file spam.py
```

```
pip3 install -r require.txt
python app.py
```
